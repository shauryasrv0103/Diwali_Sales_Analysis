# Diwali_Sales_Analysis

### Diwali Sales Analysis using Python
This project focuses on analyzing **Diwali sales data** to uncover customer behavior and business insights.  

- **Libraries Used**:  
  - NumPy → numerical computations  
  - Pandas → data cleaning & preprocessing  
  - Matplotlib & Seaborn → data visualization  

- **Analysis Performed**:  
  - Cleaned and structured raw sales data for consistency.  
  - Conducted **exploratory data analysis (EDA)** to study demographics, spending patterns, and product categories.  
  - Visualized trends across **age, gender, marital status, occupation, and region**.  
  - Compared product preferences across categories like **Food, Clothing, and Electronics**.  

- **Key Insights**:  
  - Married women aged **26–35 years** from **Uttar Pradesh, Maharashtra, and Karnataka**.  
  - Working in **IT, Healthcare, and Aviation sectors**.  
  - More likely to purchase **Food, Clothing, and Electronics** during the festive season.  

- **Business Impact**:  
  - Helps retailers design **targeted marketing campaigns**.  
  - Provides actionable insights for **inventory planning** and **customer segmentation**.  
  - Demonstrates how Python can be leveraged for **data‑driven decision making**.
